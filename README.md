# Domain Safety Checker

An automated domain safety checker built with n8n and VirusTotal API.

## How it works
1. User submits a domain via Tally form
2. n8n sends the domain to VirusTotal API
3. Result is sent to user's email automatically

## Tools Used
- n8n (automation)
- VirusTotal API (domain analysis)
- Tally.so (form)
- Gmail (notification)

## Features
- Automated domain reputation check
- Email notification with results
- No manual intervention required
